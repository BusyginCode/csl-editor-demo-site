To use in your reference manager


Mendeley

Mendeley has its own instance of the CSL Editor hosted in http://csl.mendeley.com. If you use that CSL Editor instance Mendeley will host your style, install automatically to your machine and help to share the style with other CSL users.

If you want install the current editted style in your Mendeley Desktop 1.7 or above follow these steps:

  • Save the style
  • In Mendeley Desktop, go to the View menu, Citation Style, More Styles...
  • Drag and drop the saved style into the style list

Zotero

Use the above button to save your .csl file wherever you like, then:

  • Firefox plugin

    Drag the saved file into any open Firefox window.

  • Standalone

    In Zotero standalone go to Preferences->Cite tab->Styles, and click the '+' button to add your new style.

Papers 2

Use the above button to save your .csl file to the 'Styles' subfolder within your Papers2 library folder. More info.
